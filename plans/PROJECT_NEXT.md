## Реорганизация файлов плана

Все файлы планов живут в **`plans/`** в корне проекта (отслеживаются в git).
Глубже папка → детальнее уровень.

```
plans/
├── PROJECT_DONE.md                   # архив завершённых Phase 0..1.7
├── PROJECT_NEXT.md                   # этот файл (активный план)
└── novel/
    ├── README.md                     # обзор всего проекта книги + tree map
    ├── 00-bible/                     # File 1 — структурная база
    │   ├── 00-overview.md            # сводка мира + персонажей + тона
    │   ├── 01-world.md               # детально: город, год, тех, ПДД‑контекст
    │   ├── 02-characters.md          # V, Мира, Лика, антагонисты, второстепенные
    │   ├── 03-themes.md              # темы, тон, мотивы, лейтмотивы
    │   ├── 04-corporations.md        # CP2077‑канон + российские корпы
    │   ├── 05-gangs.md               # Скоморохи / Сети Кащея / Гуляйполе и др.
    │   ├── 06-pdd-mapping.md         # таблица 26 тем ↔ номер главы
    │   ├── 07-glossary.md            # in‑world термины
    │   └── 08-economy.md             # валюта (эди €$), Trauma Team‑подписки, чёрный рынок, штрафы ГИБДД
    ├── 01-plot/                      # File 2 — сюжет
    │   ├── 00-overview.md            # 3‑актовая структура + romance beats
    │   ├── 01-act-1.md               # Act I «Знакомая» (главы 1‑8)
    │   ├── 02-act-2.md               # Act II «Подруга» (главы 9‑16)
    │   └── 03-act-3.md               # Act III «Любимая» (главы 17‑22)
    ├── 02-chapters/                  # File 3 — поглавная разбивка
    │   ├── 00-overview.md            # карта 22 глав с синопсисами
    │   ├── ch-01.md                  # ch01: сцены, бит, ПДД‑правила, диалоги
    │   ├── ch-02.md
    │   ├── ...
    │   └── ch-22.md
    └── 03-book/                      # File 4 — сама проза (MDX)
        ├── ch-01.mdx
        ├── ch-02.mdx
        ├── ...
        └── ch-22.mdx
```

**Astro Content Collection** настраивается через `loader: glob({...base: '../plans/novel/03-book'})`
чтобы рендерить главы из `plans/novel/03-book/*.mdx` — таким образом «книга
в глубине» как вы хотите, но Astro её рендерит как контент.

## Phase 1.8 — Киберпанк‑роман

### Концепт (утверждён)

| Элемент | Решение |
|---|---|
| Главный герой | **V** — никнейм, настоящее имя «Вадим» упоминается 1‑2 раза |
| Героиня | **Мира** — Judy‑like: тонкая, татуировки, мастер кибер‑протезов, мягкий низкий голос, упрямство в принципах |
| Сеттинг | Вымышленный российский мегаполис (имя в bible) |
| Вселенная | **Общая с Night City (CP2077)** — те же мегакорпы существуют глобально |
| Арка | Знакомая → подруга → любимая, через череду «рейсов» |
| Жанр / тон | Tech‑noir + romance + action; меланхолия + нежность + резкие боевые вспышки |
| Конфликт | **Биорезерв** (РФ биокорп) проводит сделку с **Militech** по нейропрототипу; Лика (сестра Миры) слила сделку → её убрали; теперь обе охотятся за резервом, который у Миры |
| Роль ПДД | Знание = выживание: дроны ГИБДД, мгновенные штрафы, 3 нарушения/сутки = блок прав. «Чистый» статус курьера = валюта |
| Объём | ~22 главы вразнобой: большинство 1 тема ПДД, некоторые 2, **3 главы без ПДД** (чистый эмоциональный/сюжетный эпизод) |

### Экономика и канон CP2077 (08-economy.md)

| Элемент | Деталь |
|---|---|
| **Валюта** | **Эди** (€$, Eurodollar) — каноническая валюта CP2077. Полная форма «евродоллар», в живой речи «эди» / «эдди». Курс к рублю — в bible (~1 эди = 80 ₽ по чёрному рынку) |
| **Trauma Team** | Платная подписка‑скорая. Тиры (Bronze/Silver/Gold/Platinum) определяют скорость прилёта. V — Bronze (никогда не платил Platinum). Мира — без подписки. Важный сюжетный рычаг в боевых сценах |
| **Штрафы ГИБДД** | Списываются мгновенно с импланта‑чипа в эди. Тарифы из реальных КоАП РФ конвертированы (превышение 60 км/ч → ~1500 эди ≈ 5000 ₽). 3 нарушения/сутки → блок прав на чипе |
| **Чёрный рынок** | «Чистый» статус курьера = валюта (за 1 «чистый» месяц курьер может купить новые импланты). Расценки и кулисы — в bible |
| **Импланты** | Каноническая шкала Humanity (CP2077): чем больше железа, тем выше риск киберпсихоза. Скоморохи — на пороге. V обычный, Мира 1‑2 кибер‑имплант |
| **Net** | Old Net (хаос, после DataKrash 2022) ↔ New Net (безопасный, корпы). Сети Кащея ходят по Old Net |
| **Эпоха** | 2087 (на 10 лет позже событий CP2077). Аура «Time of the Red» уже устаканилась, но всё ещё много следов Четвёртой корпоративной войны |

### Маппинг ПДД → главы (черновик, финал в bible)

22 главы покрывают 26 тем ПДД (некоторые главы — 2 темы, 3 главы — без ПДД).

```
Act I — экспозиция, базовые правила (главы 1‑8)
  1. Первый рейс                  → Общие положения
  2. Знаки на дожде               → Дорожные знаки (предупреждающие)
  3. Право первого                → Знаки приоритета
  4. Зона недоступа               → Запрещающие знаки
  5. Предписание + Особые         → 2 темы
  6. Информационные + Сервис      → 2 темы
  7. Разметка                     → Дорожная разметка
  8. Дождь в Аоиро                → БЕЗ ПДД (флешбэк, разговор о Лике)

Act II — усложнение (главы 9‑16)
  9. Регулировщик                 → Светофоры / регулировщик
  10. Первый поворот наедине      → Начало движения, маневрирование
  11. Полоса                      → Расположение ТС на проезжей части
  12. Скорость                    → Скорость движения
  13. Погоня                      → Обгон / встречный (экшен)
  14. Парковка у клиники          → Остановка и стоянка
  15. Перекрёсток с приоритетом   → Перекрёстки + приоритет (2 темы)
  16. Шрам                        → БЕЗ ПДД (перестрелка, V спасает Миру)

Act III — кульминация (главы 17‑22)
  17. Переход                     → Пешеходные переходы и маршрутные ТС
  18. Рельсы                      → Ж/д пути (опасность)
  19. Магистраль и жилзона        → 2 темы
  20. Свет и сирена               → Маршрутные ТС + внешние приборы (2 темы)
  21. Буксир и обучение           → Буксировка + учебная езда (флешбэк) (2 темы)
  22. Финальный КПП               → Перевозка людей+грузов + доп. требования + тех. состояние (всё вместе, финал)
   — Эпилог: БЕЗ ПДД, V и Мира за стеной города
```
