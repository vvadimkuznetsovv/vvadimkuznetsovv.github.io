# Cyberpunk Novel · план книги

Структура каталога — **глубже → детальнее**.

```
novel/
├── README.md                    ← этот файл, обзор + tree map
├── 00-bible/                    File 1 — что должно быть КАНОНОМ
│   ├── 00-overview.md           сводка: 1‑страничный summary
│   ├── 01-world.md              сеттинг: год, город, технологии
│   ├── 02-characters.md         персонажи: V, Мира, Лика, антагонисты, второстепенные
│   ├── 03-themes.md             темы, тон, лейтмотивы
│   ├── 04-corporations.md       CP2077‑канон + новые российские
│   ├── 05-gangs.md              банды и фракции
│   ├── 06-pdd-mapping.md        26 тем ПДД ↔ номер главы
│   ├── 07-glossary.md           in‑world термины
│   └── 08-economy.md            валюта (эди €$), Trauma Team, штрафы, чёрный рынок
│
├── 01-plot/                     File 2 — что должно ПРОИЗОЙТИ
│   ├── 00-overview.md           3 акта + romance beats + action beats
│   ├── 01-act-1.md              «Знакомая» — главы 1‑8 синопсисы
│   ├── 02-act-2.md              «Подруга» — главы 9‑16 синопсисы
│   └── 03-act-3.md              «Любимая» — главы 17‑22 + эпилог
│
├── 02-chapters/                 File 3 — КАК это пишется поглавно
│   ├── 00-overview.md           карта 22 глав
│   ├── ch-01.md                 сцены, диалоги, ПДД‑компоненты, тон
│   ├── ch-02.md
│   ├── ...
│   └── ch-22.md
│
└── 03-book/                     File 4 — САМ ТЕКСТ (MDX)
    ├── ch-01.mdx
    ├── ch-02.mdx
    ├── ...
    └── ch-22.mdx
```

## Порядок написания

1. **Сначала** заполняется `00-bible/` (фундамент канона)
2. **Потом** `01-plot/` (опираясь на bible)
3. **Потом** `02-chapters/` (опираясь на plot, поактно)
4. **Потом** `03-book/` (опираясь на breakdown, поглавно)

На каждом уровне — утверждение пользователем перед переходом к следующему.

## Технически

- `00`/`01`/`02` — обычный Markdown, читается людьми
- `03-book/*.mdx` — MDX с компонентами для рендеринга в сайт
- Astro Content Collection (`src/content/config.ts`) использует custom loader,
  читающий из `plans/novel/03-book/` через glob с относительным base
